# NoteShift: PDF Score Transposer 🎼

> 🚧 **Статус проекта: В процессе разработки (In Progress)** 🚧
> На данный момент реализована базовая архитектура, работает основной функционал нотного архива, и ведется активная интеграция модулей обработки музыкального текста. Проект регулярно обновляется.

## 🎯 О проекте
**NoteShift** — это специализированная многопользовательская веб-платформа, решающая одну из самых трудоемких задач для музыкантов: автоматическое транспонирование нот. 

Главная цель — реализовать бесшовный процесс: 
**Загрузка PDF → Распознавание нотного текста (OMR) → Математический сдвиг тональности → Выдача готового PDF.**

В текущей версии система обеспечивает надежное хранение данных о партитурах, композиторах и категориях, а также предоставляет инструменты для персонализации и аналитической отчетности.

## 🛠 Технологический стек
* **Backend:** Java 21, Spring Boot 4.0.3, Maven.
* **Database:** PostgreSQL, Hibernate (Spring Data JPA).
* **Security:** Spring Security (RBAC: USER/ADMIN), шифрование паролей через BCryptPasswordEncoder.
* **Frontend:** Thymeleaf (Server-side rendering).
* **REST API:** Spring Data REST, Swagger/OpenAPI.
* **Testing:** Mockito (Unit), RestAssured (API), Selenium WebDriver (UI).
* **Concurrency:** CompletableFuture для асинхронной обработки тяжелых задач.

## 🚀 Текущий функционал (реализовано)
* **Авторизация и безопасность:** Вход в систему по логину и паролю, разграничение прав на основе ролей (USER, ADMIN).
* **Каталог и поиск:** Просмотр списка всех партитур через веб-интерфейс и поиск произведений по названию, сложности и автору.
* **Управление данными:** CRUD-операции над справочниками композиторов и категорий (доступно администратору).
* **Персонализация:** Возможность сохранять партитуры в личный список избранного с добавлением рейтинга и заметок.
* **Отчетность:** Запуск асинхронной генерации HTML-отчетов по статистике системы с использованием параллельных потоков.

## 🗺 (Roadmap до PDF-to-PDF)
Данные функции заложены в архитектуру на уровне сервисов и планируются к реализации в следующих релизах:
* [ ] **Автоматическое распознавание PDF-файлов** через Audiveris (OMR).
* [ ] **Математическое транспонирование нот** в формате MusicXML.
* [ ] **Экспорт транспонированных партитур** в PDF через MuseScore.
